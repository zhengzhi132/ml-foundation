# ml-foundation

机器学习基础学习项目。

## 目录结构

```
├── python-basics/       # Python 基础语法速通
│   ├── ipynb/           # Jupyter Notebook 教程文件
│   │   ├── 01-variables-and-types.ipynb   # 变量与数据类型
│   │   ├── 02-collections.ipynb           # 列表、元组、字典、集合
│   │   ├── 03-control-flow.ipynb          # 条件与循环
│   │   ├── 04-functions.ipynb             # 函数
│   │   ├── 05-file-io.ipynb               # 文件读写
│   │   └── 06-string-processing.ipynb     # 字符串处理
│   ├── CounterWords.py      # 单词统计示例程序
│   ├── ContactsManangeSystem.py  # 通讯录管理系统示例
│   ├── input.txt            # 单词统计输入文件
│   └── README.md            # Python 基础学习指南
└── README.md                # 本文件
```

## 前置要求

- Python 3.x
- Jupyter Notebook（用于运行 .ipynb 文件）
