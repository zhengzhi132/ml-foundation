# Python 基础语法速通

本教程面向零基础或刚入门的 Python 学习者，目标是 **快速掌握 Python 最核心的日常用法**。

## 学习目标

- ✅ 能写函数
- ✅ 能处理字符串和列表
- ✅ 能读写文件
- ❌ 不碰任何数学和第三方库

## 学习路线（建议按顺序）

| 文件 | 内容 | 预计用时 |
|------|------|----------|
| `01-variables-and-types.md` | 变量、数据类型（int/float/str/bool） | 15 min |
| `02-collections.md` | 列表、元组、字典、集合 | 25 min |
| `03-control-flow.md` | if/elif/else、for、while | 20 min |
| `04-functions.md` | 函数定义、参数、返回值、作用域 | 20 min |
| `05-file-io.md` | 文件读写、with 语句 | 15 min |
| `06-string-processing.md` | split、join、replace、f-string | 15 min |

> **建议**：打开一个 Python 交互式环境（`python` 或 `ipython`），边看边敲代码，效果最好。

## 前置要求

- 电脑上安装了 Python 3.x（可在终端执行 `python --version` 确认）
- 一个文本编辑器（记事本都行，推荐 VS Code）
